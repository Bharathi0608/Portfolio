# Interactive Professional Portfolio

A premium, high-performance portfolio website built for a Full Stack Developer. This project features advanced CSS animations, an interactive particle background, 3D tilt effects, and a responsive design tailored for recruiters.

## 🚀 Live Demo
Check out the live version here: [https://portfolio-bharathi-six-6.vercel.app/]

## 🛠️ Tech Stack
- **Frontend**: HTML5, Vanilla CSS, JavaScript (ES6+)
- **Interactions**: 
  - [Vanilla-Tilt.js](https://micku7zu.github.io/vanilla-tilt.js/) for 3D card effects.
  - Custom Canvas Particle System.
  - Scroll Reveal animations.
- **Icons**: [FontAwesome](https://fontawesome.com/) & [Devicon](https://devicon.dev/) for tech logos.
- **Typography**: Inter (Google Fonts).

## ✨ Key Features
- **Dynamic Interaction**: 
  - Interactive constellation background that reacts to mouse movement and clicks.
  - Magnetic button physics for a premium UI feel.
  - Smooth typing animation for hero roles.
- **Glassmorphic UI**: Modern dark-themed design with blurred surfaces and vibrant cyan accents.
- **Responsive Layout**: Fully optimized for Desktop, Tablet, and Mobile devices.
- **Recruiter Focused**: Clear sections for Internships, Education, Skills, and Projects with high readability.

## 📦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) installed on your machine.
- A modern web browser.

### Installation & Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Bharathi0608/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
